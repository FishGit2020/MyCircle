## Dev Setup

```bash
# Install dependencies (from repo root)
pnpm install

# Build shared package (required before running MFE tests)
pnpm build:shared

# Start local dev with Firebase emulator
pnpm dev
```

---

## Build After Schema Changes

```bash
# 1. Edit functions/src/schema.ts
# 2. Run codegen to regenerate types
pnpm codegen

# 3. Rebuild shared
pnpm build:shared

# 4. Verify backend types compile
cd functions && npx tsc --noEmit && cd ..
```

## Implementation Order

Follow the priority order from the spec. Each priority level is independently deployable:

| Step | Feature | Backend Changes? | Key Files |
|------|---------|-----------------|-----------|
| 1 | Search & Filter | No | `SearchFilterBar.tsx`, `CloudFiles.tsx` |
| 2 | Folder Organization | Yes | `cloudFilesEnhancements.ts`, `useFolders.ts`, `FolderList.tsx`, `FolderBreadcrumb.tsx` |
| 3 | File Preview | No | `FilePreviewModal.tsx`, `FileCard.tsx` |
| 4 | Rename Files | Yes | `renameFile` mutation + resolver, `FileCard.tsx` |
| 5 | Targeted Sharing | Yes | `shareFileWith` / `revokeFileAccess` mutations, `useTargetedSharing.ts`, `ShareRecipientsModal.tsx` |
| 6 | Storage Quota | No | `StorageQuotaBar.tsx`, `CloudFiles.tsx` |
